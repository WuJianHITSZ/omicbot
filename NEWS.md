# omicbot 0.1.2

## New features

- Added `save_chat()` and `resume_chat()` for persisting and restoring Omicbot chat sessions.
- Added Xiaomi MiMo provider support through an OpenAI-compatible endpoint.
- Added example gene expression data and violin plot scripts for omics visualization workflows.
- Added VS Code and Positron keybinding templates in `config/`.

## Improvements

- Updated package metadata and README documentation for the 0.1.2 release.
- Improved `settings()` cancellation handling.
- Moved in-package help text to `help.md`.
