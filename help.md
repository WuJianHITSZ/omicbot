# Omicbot

## Help

### 1) Before You Start
Set a keyboard shortcut for the RStudio addin named `omicbot` at first (tutorial below).
After the shortcut is set, type your prompt in the Console pane as if it were a normal R command, then press the shortcut (not Enter). Omicbot will receive your prompt and response.

How to set the keyboard shortcut for Omicbot:
- Open in RStudio **Tools -> Modify Keyboard Shortcuts**
- Search for **omicbot**
- Set your shortcut (`Shift+Enter` is strongly recommended.)

### 2) Ask Omicbot to Diagnose Console Error
If you run into an error in the console, type `??` and trigger the Omicbot shortcut.
Omicbot will capture error context and provide troubleshooting assistance.

### 3) Send Multi-line Prompts from Clipboard
If your prompt contains multiple lines (for example, copied from another source):
- Copy the full text to clipboard
- Return to the RStudio Console
- No need to type anything but make sure the caret (blinking vertical line) is in the Console pane
- Trigger the Omicbot shortcut
Omicbot will know your prompt from clipboard and response.
